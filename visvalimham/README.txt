Para ejecutar la interfaz grafica desde Linux
    acceda a la carpete Ejecutable
    abra una terminal en esta direccion
    Teclee
    -> ./InterfazGrafica

Para probar las funciones fuera de la terminal
    acceda a ala carpeta "Archivos C++"
    Detro de main_seg.cpp puede poner el nombre de su imagen en el constructor de Segmentation
    Guarde y Teclee desde la terminal
    -> g++ main_seg.cpp -o test
    -> ./test
